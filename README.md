# Hand Breakout 🎮

A game controlled by hand gestures in the browser.

## Features

- Camera-based hand tracking with MediaPipe Hands
- Move your hand left/right to control the paddle
- Open hand to start or resume
- Fist to pause
- Canvas-based breakout gameplay
- Score system and multi-level progression

## Controls

- Move hand horizontally: move paddle
- Open hand: start / resume
- Fist: pause
- Space key: pause / resume fallback

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Canvas 2D API
- MediaPipe Hands

## Local Run

Open `index.html` directly in a browser, or serve the folder with a local static server for more reliable camera permissions.
